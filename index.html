CS 101 - HTML, HTML5, CSS and CSS3

HTML and CSS are fundamental building blocks for creating interactive and stylish websites. Once you learn this, you can learn any other backend programming languages (PHP, Ruby, Python, etc) to create a database driven website. This course should take approximately 20-40 hours to master and assumes that you do not have any prior programming experience. At the end of the course, you will know how to create a website from scratch!

CS 110 - jQuery and jQueryUI

Pre-requisites: CS 101

jQuery and jQueryUI are javascript libraries to make your website very interactive. It's also cross-browser friendly, which means you can make very cool interactive things without having to worry about other browsers (mainly Internet Explorer) crashing or not working properly. After learning jQuery, jQueryUI, you will know how to do drag and drop, have html elements fade in/out, have fancy pop up dialogs open, and many other cool things.

You should have a solid understanding of HTML and CSS before jumping into this course. Estimated time of completion: 20-40 hours.
